# Topic 1.2 - Student Profile as an SFC

## Goal

This exercise turns a simple screen into a readable single-file component with template, script setup, and scoped styles living together.

## What you are training

- keep state and template in the same SFC without losing structure
- use <script setup> as the default Vue 3 authoring style
- read and edit a component from top to bottom

## Task

1. Use studentProfile inside App.vue.
2. Render the student name, role, current track, learning goal, skill focus, and button label.
3. Keep the component as a real SFC with <script setup> and <style scoped>.

## Run locally

```bash
npm install
npm run test
```
