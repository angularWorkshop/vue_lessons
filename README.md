# Сценарный тест с сетевым мокингом: проверить вход, фильтрацию и переход без живого backend

## Goal

Научиться закреплять контракт компонента или flow через тест, а не только глазами в браузере.

## What you are training

- Vitest + Vue Test Utils
- public contract testing
- reproduce bug or flow through tests

## Task

1. сделай тесты говорящими
2. проверяй публичное поведение, а не внутренности без нужды
3. оставь зелёный результат после исправления

## Run locally

```bash
npm install
npm run test
```
