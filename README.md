# Theme Panel Bindings

## Goal

Connect panel classes, styles, and labels to one theme state model.

## What you are training

- bind visual mode through a class
- bind inline styles from state
- keep texts and colors in one model

## Task

1. Make the theme panel reactive.
2. Bind the mode class.
3. Bind the panel and button styles from state.

## Run locally

```bash
npm install
npm run test
```
