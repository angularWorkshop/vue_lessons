# Cart Computed Totals

## Goal

Move subtotal, discount, total, and checkout state into honest computed values.

## What you are training

- derive values from source state through computed
- keep the template focused on ready values
- recalculate totals after quantity changes

## Task

1. Replace the hardcoded totals with computed values.
2. Keep quantity changes as the source of truth.
3. Render subtotal, discount, total, and checkout state from the computed model.
