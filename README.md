# Typed форма профиля: собрать переиспользуемую модель редактирования без any

## Goal

Усилить Vue-контракт через TypeScript: не спрятать смысл под типами, а сделать входы и выходы компонента честными.

## What you are training

- script setup lang="ts"
- typed props/emits or generic helpers
- no any in the public contract

## Task

1. добавь TypeScript там, где это усиливает контракт
2. опиши входные данные и результат явно
3. не используй any в публичной части решения

## Run locally

```bash
npm install
npm run test
```
