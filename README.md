# Topic 1.2 - Split the Workspace into SFC Files

## Goal

This exercise shows that a Vue screen does not have to live in one huge file: even the first workspace can be split into several focused components.

## What you are training

- read the root component as a composition point
- work with several SFC files inside one small feature
- understand how a component tree becomes a screen

## Task

1. Keep App.vue as the root composition point with WorkspaceHero and WorkspaceChecklist.
2. Finish both child components so they render real data from workspace-overview.js.
3. Make the composed screen match the expected title, texts, checklist items, and CTA button.

## Run locally

```bash
npm install
npm run test
```
