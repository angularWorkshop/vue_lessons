# Modal v-if and v-show

## Goal

Separate modal frame visibility from modal content existence.

## What you are training

- pick between v-if and v-show by behavior
- keep frame and content roles separate
- test a modal flow through user actions

## Task

1. Keep the modal frame mounted through v-show.
2. Render the inner card only when content exists.
3. Make open, close, and toggle-content stay predictable.

## Run locally

```bash
npm install
npm run test
```
