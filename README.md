# Registration Form v-model

## Goal

Build a basic form flow around v-model and one readable form state.

## What you are training

- bind text inputs with v-model
- bind a checkbox to the form model
- change submit status through state

## Task

1. Connect v-model to the fields.
2. Bind the agreement checkbox.
3. Update the submit status and result message.

## Run locally

```bash
npm install
npm run test
```
