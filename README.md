# Search Box v-model

## Goal

Keep a search child connected to the parent through a clear one-way contract.

## What you are training

- describe modelValue and update:modelValue in a child
- use v-model on a custom component
- filter a parent list from child input

## Task

1. Add the child props and emits contract.
2. Emit the new query from the input.
3. Use the child through v-model in App.vue.

## Run locally

```bash
npm install
npm run test
```
