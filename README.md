# Operations Workspace SPA: собрать маршруты, список и редакторский flow в один экран

## Goal

Собрать маленький workspace с маршрутом списка и маршрутом редактирования внутри одного SPA-потока.

## What you are training

- compose one small router-driven workspace
- keep list and editor connected
- move through a realistic user flow

## Task

1. сделай список маршрутом по умолчанию
2. добавь переход в editor-маршрут по id
3. оставь layout и навигацию читаемыми

## Run locally

```bash
npm install
npm run test
```
