# Из плоских маршрутов в иерархию: перестроить router под parent-child структуру

## Goal

Перестроить плоские маршруты в parent-child структуру, где layout и дочерние экраны читаются как одно дерево.

## What you are training

- nested routing as structure
- read router tree as UI tree
- separate layout and child pages

## Task

1. создай parent route для reports
2. перенеси detail/view в children
3. оставь layout видимым вместе с дочерней страницей

## Run locally

```bash
npm install
npm run test
```
