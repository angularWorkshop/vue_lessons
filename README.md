# Data card со слотами: дать гибкость header, actions, content и empty-state без десяти props

## Goal

Сделать UI-композицию гибкой и читаемой за счёт слотов и специальных Vue-механик, а не через десять жёстких props.

## What you are training

- slots or advanced rendering patterns
- keep flexible component contracts readable
- test real UI behavior

## Task

1. сделай публичный контракт компонента явным
2. используй slots там, где шаблон должен оставаться гибким
3. убедись, что поведение видно прямо в тестах

## Run locally

```bash
npm install
npm run test
```
