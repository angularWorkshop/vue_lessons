# Reactive Product Card

## Goal

Build the first honest local state model in Vue with ref and reactive.

## What you are training

- use `ref` for one standalone value
- use `reactive` for one linked screen model
- render the screen from real state instead of TODO text

## Task

1. Create `quantity` with `ref`.
2. Create `productState` with `reactive`.
3. Render the real product card texts and discounted total.

## Run locally

```bash
npm install
npm run test
```
