# Presentational Offer Card

## Goal

Keep an offer card focused on display, not on hidden pricing rules.

## What you are training

- recognize business logic leaking into a child component
- prepare view data in the parent
- render a card through clear props

## Task

1. Remove the policy import from the child card.
2. Prepare the view data in App.vue.
3. Pass badge, priceText, and tone into the child card.

## Run locally

```bash
npm install
npm run test
```
