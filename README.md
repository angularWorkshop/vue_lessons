# Dropdown Event Modifiers

## Goal

Use event modifiers to keep an interaction flow readable.

## What you are training

- use .prevent, .enter, and .stop
- describe keyboard flow in the template
- prevent nested clicks from leaking to the parent

## Task

1. Add the missing event modifiers.
2. Keep the dropdown open on the inner button click.
3. Make Enter open the dropdown.

## Run locally

```bash
npm install
npm run test
```
